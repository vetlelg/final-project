GitHub rules

Workflow

  • Make a new temporary branch for every change you make (Remember to pull origin before you create a new branch).
  • Create a pull request when you want to merge. If there is no conflict, continue to merge. Merge-master Vetle will take care of any conflict.
  • When the merge is complete, delete the temporary branch.

CSS structure

  • A global.css file for elements all HTML files use.
  • Every HTML file has its own CSS file for elements only used in that particular HTML file.
  • Remember to comment everything! It is easier for everyone else if you do.
